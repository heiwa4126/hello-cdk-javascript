# hello-cdk-javascript

Bun と ECMAScript にした
[AWS CDK v2 チュートリアル](https://docs.aws.amazon.com/ja_jp/cdk/v2/guide/hello_world.html)。

## 手順

```sh
bun i
bunx cdk list
bunx cdk deploy
bunx cdk diff
bunx cdk destroy
```
